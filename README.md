# personal-lab

個人開発で得た知見・失敗・判断を資産として蓄積するためのリポジトリ。

## 目的
- 未完成プロジェクトを「失敗」で終わらせない
- 技術的な学びを横断的に再利用する
- 未来の自分のための研究ノートを作る

## 構成
- iOS/: iOS（Swift / SwiftUI）関連メモ
- WEB/: Web フロントエンド / バックエンド関連メモ
- Android/: Android メモ
- AI/: AI 関連メモ

## 運用ルール
- コードは置かない（別リポジトリに置く）
- 各メモには「何をしようとしたか」「どこで詰まったか」を必ず書く
- プロジェクト README からここへリンクする
